SQL

Javascript

Linguagem de programação de interatividade do site, conhecida oficialmente como Ecmascript, possuindo Java na frente ao inves de Ecma pois o fundador tentou surfar a onda da liguagem Java, que é muito famosa, mesmo sendo completamente diferente a sua sintaxe.

React

React JS é uma biblioteca JavaScript para a criação de interfaces de usuário

Criado em 2011 pelo time do Facebook, o React surgiu com o objetivo de otimizar a atualização e a sincronização de atividades simultâneas no feed de notícias da rede

Git e GitHub

O Git e o GitHub são ferramentas de versionamento de código, sendo o ultimo mais uma rede social, utilizados por empresas para armazenar e atualizar o software em desenvolvimento.
